# Repository Minimum

A bootstrapped repository is considered minimally initialized when it contains:

- `AGENTS.md`
- `AGENT_INDEX.md`
- `PROGRESS.md`
- `MEMORY.md`
- `NEXT_STEP.md`
- `.harness/bootstrap.toml`
- `docs/project/README.md`
- `docs/architecture/overview.md`
- `docs/architecture/layers.md`
- `docs/testing/strategy.md`
- `docs/superpowers/templates/SPEC_TEMPLATE.md`
- `docs/superpowers/templates/PLAN_TEMPLATE.md`
- `docs/superpowers/templates/CHANGE_REQUEST_TEMPLATE.md`
- `docs/superpowers/templates/EVIDENCE_TEMPLATE.md`

Missing any of these means the repository is not ready for stable agent work.
