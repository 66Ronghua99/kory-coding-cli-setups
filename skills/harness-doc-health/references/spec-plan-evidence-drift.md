# Spec-Plan-Evidence Drift

Drift appears when the graph is still linked but the truth has aged out.

## Required Chain

- spec defines intent truth
- plan defines execution truth
- evidence defines verification truth

## Common Drift Signals

- acceptance criteria changed but plan claims did not
- plan changed but evidence still cites old claims
- evidence exists but no longer verifies the active plan
- an older plan remains `active` after its loop is closed
