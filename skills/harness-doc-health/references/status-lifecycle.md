# Status Lifecycle

Use document status to reflect reality, not aspiration.

## Recommended Meanings

- `draft`: being prepared, not the live truth yet
- `active`: current source of truth for an open loop
- `superseded`: replaced by a newer active document
- `deprecated`: still present but intentionally on the way out
- `archived`: closed historical record

## Rules

- A closed loop should not leave old plans marked `active`.
- A superseding document should be easy to follow via metadata and text references.
- Historical evidence may stay linked, but it should not be mistaken for current verification.
