# Daily Plan 模板参考

> 此文件为 agent 提供模板参考，请勿手动修改

## 文件位置

模板源：`_template/daily.md`

## Frontmatter 格式

```yaml
---
title: "YYYY-MM-DD Daily Plan"
date: YYYY-MM-DD
type: diary
status: draft
tags: [daily]
mood:
summary: "今日核心主题一句话"
---
```

## 必填章节

1. **> 引用句** - 今日核心主题/目标一句话
2. **## 今日 TODO**
   - ### 优先级 P0
   - ### 优先级 P1
   - ### 优先级 P2
3. **## 今日复盘**
   - ### 今日收获
4. **## 碎片灵感**

## 文件命名

- 格式：`YYYY-MM-DD.md`
- 示例：`2026-03-09.md`
- 目录：`_daily/`
